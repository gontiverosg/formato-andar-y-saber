Trabajar en Overleaf (Recomendado)
1.	Cree una cuenta en www.overleaf.com.
2.	Haga clic en "Subir" y cargue el .zip de la plantilla.
3.	Edite tu_paper_ADS.tex (observe que el estado original de tu_paper_ADS.tex genera el archivo instrucciones.pdf, por lo tanto tu_paper_ADS.tex contiene las configuraciones necesarias para crear su propio manusctrito).
4.	En el archivo referencias.bib usted encontrará instrucciones para colocar la bibliografía correspondiente de tal manera que LaTex integre a la bibliografía de su texto mientras usted escribe.
5.	Use "Recompilar" para previsualizar su nuevo PDF.
